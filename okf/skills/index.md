# Agentic Skills

Skill packages that teach AI coding agents (Claude Code, Cursor, and 40+ others,
via [skills.sh](https://github.com/vercel-labs/skills)) to drive OpenAlgo. Install
with `npx skills add <package>`.

* [Execution Skills](execution.md) - Order execution, custom algos, scanners, streaming, and alerts over the OpenAlgo SDK.
* [Indicator Skills](indicators.md) - Charting, scanners, custom indicators, and live feeds with 100+ indicators.
* [Backtesting Skills](backtesting.md) - VectorBT backtesting across Indian/US/Crypto markets with realistic costs and tearsheets.
