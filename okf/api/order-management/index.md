# Order Management

* [PlaceOrder](placeorder.md) - Place a new order with the broker
* [PlaceSmartOrder](placesmartorder.md) - Place a position-aware smart order that reconciles to a target position size
* [OptionsOrder](optionsorder.md) - Place an options order using a strike offset from ATM
* [OptionsMultiOrder](optionsmultiorder.md) - Place a multi-leg options order in a single call
* [BasketOrder](basketorder.md) - Place multiple orders simultaneously as a basket
* [SplitOrder](splitorder.md) - Split a large order into smaller chunks
* [ModifyOrder](modifyorder.md) - Modify an existing open order
* [CancelOrder](cancelorder.md) - Cancel a specific open order by order id
* [CancelAllOrder](cancelallorder.md) - Cancel all open orders
* [ClosePosition](closeposition.md) - Close all open positions
* [PlaceGTTOrder](placegttorder.md) - Place a SINGLE or OCO GTT (Good Till Triggered) order
* [ModifyGTTOrder](modifygttorder.md) - Modify an active GTT trigger
* [CancelGTTOrder](cancelgttorder.md) - Cancel an active GTT trigger
* [GTTOrderBook](gttorderbook.md) - List active GTT triggers
